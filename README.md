# Ava Expense Tracking

Ava Expense Tracking adalah aplikasi pencatatan keuangan berbasis **WhatsApp** yang memungkinkan pengguna mencatat pemasukan, pengeluaran, dan melihat laporan keuangan hanya dengan mengirimkan pesan melalui WhatsApp.

Aplikasi ini dibangun agar proses pencatatan menjadi lebih cepat tanpa perlu membuka aplikasi keuangan secara langsung.

---

## ✨ Fitur

- Mencatat pemasukan
- Mencatat pengeluaran
- Kategori transaksi
- Catatan transaksi
- Pencarian transaksi berdasarkan tanggal
- Laporan harian
- Laporan mingguan
- Laporan bulanan
- Ringkasan saldo
- Pencarian transaksi
- Integrasi WhatsApp Bot
- Integrasi dengan AI
